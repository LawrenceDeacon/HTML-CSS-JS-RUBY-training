GIT Notes

Stages of git process

  • Create new document / make changes
  • git add to stage changes
  • git commit to commit staged changes
  • git push to push staged changes to repository

Commands

  • git clone - clones a repo onto the local machine
  • git status - shows what needs to be added
  • git add - add changed items to staged, example git add * will add everything
  • git commit -m "Message" - commits staged items with a message
  • git push main - pushes commited changes to repo with a message, always use a message
  • git log - shows log of all changes
