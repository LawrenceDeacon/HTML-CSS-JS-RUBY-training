GIT Notes

Stages of git process

  • Create new document / make changes
  • git add to stage changes
  • git commit to commit staged changes
  • git push to push staged changes to repository

Commands

  • git clone - clones a repo onto the local machine
  • git status - shows what needs to be added
  • git add - add changed items to staged, example git add * will add everything
  • git commit -m "Message" - commits staged items with a message
  • git push main - pushes commited changes to repo with a message, always use a message
  • git log - shows log of all changes

Branches

Branching allows cloning of a repo with commits / changes only happening on that Branching Can merge back into main if needed

  • git branch nameOfNewBranch - creates new branch
  • git branch - with no name will show all branches for this repo
  • Asterixed branch is current
  • git checkout nameOfBranch - change to nameOfBranch
  • git checkout -b newBranch - creates and changes to newBranch
  • git merge nameOfBranch - merges nameOfBranch into current
  • Merge conflicts need to be resolved
  • git branch -d nameOfBranch - deletes nameOfBranch
