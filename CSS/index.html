Selectors

ID's and Classes

  • ID's = Add in html tag, id='xyz', in CSS with #xyz
  • Classes' = Add in html tag. class='xyz', select in CSS with .xyz

Using Selectors

  • To use selectors in CSS use div{}
  • Chain selectors .class1.class2{} , Only one element can be chained ie div.class1{} cannot use divp
  • To chose child elements use a space for example, div p {} will only select p's that are wrapped in a div

Propertys

  • Properties are put within the selectors {}
  • Properties are written as follows, property-name: value; , for example color: red; would make text go red as seen here

Flex Box

  • To create a flex box use display:flex;
  • To create a flex child use flex:x y z; Typically use flex:1;
    • x = Grow, 2 would make this item grow twice as large
    • y = Shrink, if 1 all shrink at same rate, 0 does not shrink
    • z = Basis, sets initial size before shrink or growth
    Can also use
    • flex-grow:x;
    • flex-shrink:x; 0 to not shrink
    • flex-basis:x; commonly use auto
  • 2 axis' to flex boxes, main and cross
    • Main Axis, by default is horizontal, use flex-direction: column to change to Vertical
    • Cross Axis is the opposite of Main axis
  • Justify-xxx affects the Main Axis
  • Align-xxx affects the Cross Axis
