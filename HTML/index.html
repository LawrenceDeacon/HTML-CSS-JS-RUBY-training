Notes on HTML

Tags

To write the tags below us '& l t' and '$ g t' for lesser than and greater than symboles respectivley

Opening tags are written as <div>

Closing tags are written as </div>

Some tags are self closing for example <img/>

Some types of tags commonly used

  • html - encompasses everything for html
  • head - to add scripts / link / meta data
  • body - where the content goes in the page
  • div - good for wrapping
  • p - paragraphs
  • h1 - h6 - h1 being the largest header
  • img - for adding images, includes src for where the image is and alt for alternative, ie blind
  • span - used to split out items in a string, for example the red text top of this section
  • a - hyper links, for example <a href="https://mylink.com >My Link/>
